My Skills

Here are some of my core skills and ways to learn them.

HTML5

The standard markup language for creating the structure of web pages. It organizes content.

How to Learn:

  • MDN Web Docs: An excellent resource for HTML tutorials.
  • freeCodeCamp: Offers interactive coding challenges.
  • Build Small Projects: Create your own basic webpage structures.

CSS3

Controls the styling (colors, layout, fonts) of web pages, making them beautiful and responsive.

How to Learn:

  • CSS-Tricks: Articles and tutorials on various aspects of CSS.
  • Flexbox Froggy/Grid Garden: Interactive games that teach Flexbox and Grid.
  • Style your HTML: Apply various CSS properties to HTML pages you create.

JavaScript

Makes web pages interactive and dynamic, handling animations, form validation, and data manipulation.

How to Learn:

  • The Odin Project/JavaScript.info: In-depth online courses.
  • LeetCode/HackerRank: To sharpen your problem-solving skills.
  • Add Small Features: Implement small interactive features on your website.

Tailwind CSS

A utility-first CSS framework that provides ready-to-use classes for rapid and efficient styling.

How to Learn:

  • Tailwind CSS Official Docs: The best and most comprehensive resource.
  • YouTube Tutorials: Many great videos for beginners.
  • Use in your Projects: Integrate Tailwind into existing or new projects.

React (Optional)

A JavaScript library for building user interfaces, focused on a component-based architecture.

How to Learn:

  • React Official Docs: Beginner's guides on the official website.
  • Udemy/Coursera: Paid courses for full-stack React development.
  • Build Small Components: Create small React apps like a to-do list or calculator.

Node.js (Optional)

A JavaScript runtime environment for building server-side applications.

How to Learn:

  • Node.js Official Docs: Official documentation and getting started guides.
  • Express.js: Learn a popular web framework for Node.js.
  • Build APIs: Create simple RESTful APIs that connect to databases.

UI/UX Design

Creating user interfaces and experiences that are intuitive, efficient, and enjoyable.

How to Learn:

  • Google UX Design Professional Certificate: A structured learning path.
  • Figma/Adobe XD: Learn to use UI design tools.
  • Analyze Case Studies: Study examples of good and bad design.
